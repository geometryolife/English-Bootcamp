# 辉说英语晨读

## W13D1

- Andrew: Oh, I'm so tired.
- Evelyn: Really? How come?
- Andrew: Eh, I'm working two jobs this semester.

- 安德鲁：噢，我太累了。
- 伊芙琳：真的吗？怎么了？
- 安德鲁：呃，这学期我打了两份工。

### 词汇

- I'm I am 的缩写，我是
- so 非常
- tired 疲劳的，累的
- really 是吗，真的吗
- how come 怎么了？
- working work 的现在分词，工作
- two 两个
- job 工作
- this 这个
- semester 学期
